**Table of Contents**

- [Installation](#installation)

## Installation

Currently SurfKit is available as an NPM package. It's installation is as follow:

npm: ```npm i surfkit```

bun: ```bun i surfkit```

You can also add it directly to your project like this:

npm: ```npm add surfkit```

bun: ```bun add surfkit```
